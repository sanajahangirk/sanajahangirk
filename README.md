### [📈 ICU Mortality Prediction using Vital Signs]
- Built an LSTM-based model using MIMIC-IV dataset to predict in-hospital mortality within 24h of ICU admission.
- Achieved AUROC of 0.89 with SHAP-based explanations for clinical interpretability.
- **Tech:** Python, TensorFlow, pandas, SHAP, PostgreSQL

### [💓 Arrhythmia Classification from ECG Signals]
- Processed raw 12-lead ECG signals (PhysioNet PTB-XL) and trained a CNN-LSTM hybrid model for multi-class arrhythmia detection.
- Deployed a real-time grading tool with Streamlit.
- **Tech:** WFDB, SciPy, PyTorch, Streamlit

- **Tech:** Hugging Face, PyTorch, spaCy, NLTK
